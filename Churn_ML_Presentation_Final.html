Feature Engineering

Two New Features Created

services_count
Range: 0 – 7
Sum of 7 binary service columns: MultipleLines, OnlineSecurity, OnlineBackup, DeviceProtection, TechSupport, StreamingTV, StreamingMovies.

Why: EDA showed customers with more services have higher switching cost and churn less. One numeric feature replaces 7 columns.
services_count = sum(7 binary service cols)
TenureGroup
4 Bins: 0–3
Groups tenure into 4 bands matching EDA findings:
0 = 0–12 mo  ·  1 = 13–24 mo  ·  2 = 25–48 mo  ·  3 = 49+ mo

Why: Each band shows a distinct churn rate in EDA. Uses bins=[-1,12,24,48,72] so tenure=0 falls in group 0.
pd.cut(tenure, bins=[-1,12,24,48,72], labels=[0,1,2,3])
Gender Dropped: EDA confirmed identical ~26% churn rate for Male and Female customers — adding noise-only features dilutes model signal and can reduce accuracy.
